# TSTS GENESIS v3.2 - Field Manual

### 軽トラ重心特定システム：Bluetooth接続版

このシステムは、屋根に固定した **iPhone SE (演算器)** のセンサーデータを、Bluetoothテザリングを介して運転席の **iPhone 7 (表示器)** へミラーリングし、車両の真の重心位置を特定します。

---

## 1. 事前準備（車外での作業）

* **iPhone SE**: 屋根設置用。
* **iPhone 7**: モニター用。
* **ダクトテープ**: SEを屋根に**ガチガチ**に固定するため。
* **メジャー**: 以下の2点を実測してメモ：
1. **Mount Z**: 地面から屋根（SE設置面）までの垂直距離。
2. **Mount X**: フロント車軸からSE設置位置までの前後距離。



---

## 2. 接続・設営手順（現場での儀式）

### Step 1: Bluetoothリンクの確立

1. **SE側**: 「設定」＞「インターネット共有」を **ON**（「互換性を優先」もON推奨）。
2. **7側**: Bluetooth設定からSEに接続。
* 7側の画面上部（またはコントロールセンター）に**鎖のマーク（テザリング中）**が出ていることを必ず確認。



### Step 2: SE (屋根側) の配信開始

1. ブラウザでシステムを開く。
2. **4-Digit Passcode**: 任意の4桁（例: `7777`）を入力。
3. **Mount Z / X**: 1で測った実測値を入力。
4. **Initial Guess**: 軽トラなら  /  程度を入力。
5. **「1. SE 屋根配信開始」**をタップ。ステータスが `ONLINE` になったら、屋根の指定位置にガムテープで強固に固定する。

### Step 3: iPhone 7 (室内側) の受信開始

1. ブラウザでシステムを開く。
2. SEと同じ **Passcode** を入力。
3. **「2. 7 室内モニター接続」**をタップ。
4. 数値が同期され、メインモニターに切り替わるのを確認。

---

## 3. 実走計測メソッド

* **重心高 (h) の特定**: 時速20km程度で、**一定のリズムで左右にウェービング**を行う。
* **前後位置 (x) の特定**: 直線で**緩やかな加減速**を行い、ピッチングを誘発させる。
* **数値の確定**: 7の画面で数値が安定（収束）したと思ったら、**「RESULT CAPTURE」**をタップ。表示が固定されたら、その数値を記録。

---

## 4. 現場トラブル解決（BT版特有）

* **接続が途切れたら**:
* BTテザリングは距離や遮蔽物で稀に瞬断します。7側のブラウザを**リロード**し、再度「2. 受信開始」を押してください（SE側は屋根で動いたままでOK）。


* **数値が異常に跳ねる**:
* 屋根のSEがテープの中で動いています。固定をやり直してください。


* **数値が全く変わらない**:
* ブラウザがモーションセンサーの利用を許可していない可能性があります。リロードして「許可」ボタンを押してください。



---

## 5. 安全上の注意

* 計測は安全な私有地や広い駐車場で行ってください。
* 屋根からのiPhone脱落は後続車にとって非常に危険です。テープの粘着力は念入りに確認してください。
